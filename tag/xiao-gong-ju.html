tmux cheatsheet

if you use set mouse off, then you could use system mark and copy, or if you are in Mac, you could use Option + Mouse Select

按键绑定

C-b         发送 Ctrl-b 按键
C-z         暂停（suspend） tmux 客户端

窗口创建与管理

!           把当前分区独立出来作为单独的窗口
"           横向分隔，变成上线两个分区
%           纵向分隔，变成左右两个分区
&           关闭当前窗口（window），也就是所有的分区                    
.           Prompt for an index to …
