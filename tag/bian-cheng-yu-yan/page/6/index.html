Python 3 中的 Enum

enum 模块是 Python 3 中新引入的一个用于定义枚举常量的模块。

基本使用

>>> from enum import Enum
>>> class Color(Enum):
...     RED = 1
...     GREEN = 2
...     BLUE = 3
...
>>> print(Color.RED)
Color.RED
>>> type(Color.RED)
<enum 'Color'>
>>> isinstance(Color.GREEN, Color)
True

# 可以遍历
>>> for color in Color:
...     print color
...

Color.GREEN
Color.BLUE

lua coroutine

Lua 的协程是非对称的协程也就是 resume 和 yeild 相当于调用和返回

Lua中携程相关的函数都放在coroutine包中 coroutine.create(function) 返回一个thread类型的值表示一个协程，并且处于suspend状态。 resume(co, params…) 执行一个协程，并且能够传递参数，返回运行的状态的函数yield返回的结果
