Python 中的异常总结

User defined class

for user defined exceptions, just subclass Exception

Catching Exceptions

except Exception as e: # SystemExit, KeyboardInterrupt, GeneratorExit is not captured
    log('Reason', e)    # the baseline is to record

容易抛出异常的地方

  • 网络读取，requests.get
  • 调用其他命令超时，subprocess.call
  • 读取他人的数据，IndexError
