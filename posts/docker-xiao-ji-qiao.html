docker 小技巧

删除所有停止的容器

docker rm `docker ps -aq`

删除所有没有 tag 的镜像

docker rmi `docker images | grep "^&lt;none&gt;" | awk '{print $3}'`

进入运行中的镜像

docker exec -it CONTAINER_NAME bash
